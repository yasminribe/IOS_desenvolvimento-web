Exemplo 2:

            
Duas colunas são criadas sendo a segunda com o dobro do tamanho da primeira. FR é uma unidasde fracional. 
O tamanho do conteúdo é respeitado, ou seja, se o conteúdo na primeira coluna for maior que o da segunda, a primeira será maior.

            .grid {
                display: grid; 
            }
            .grid-template-columns-2{
                grid-template-columns: 1fr 2fr;
            }

grid-template-columns: 1fr 2fr;

Exemplo 3:

            
Minmax define um valor minimo e maximo para colunas, três colunas de são criadas, a primeira tera no minimo 200px de largura e no maximo 1fr 
(isso significa que após 200px ele se expande da mesma forma que as outras colunas). As outras duas colunas vão ter 1fr. 

            .grid {
                display: grid; 
            }
            .grid-template-columns-3{
                grid-template-columns:minmax(200px , 1fr) 1fr 1fr;
            }
