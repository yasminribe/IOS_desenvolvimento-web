propriedade grid-templete-areas

            
A propriedade grid-templete-areas define um modelo de grid referenciando os nomes das areas do grid que são especificadas pela propriedade grid-area.
Repetir o nome de uma area grid faz com que o conteudo se espalhe por essas células com nome repetido. Um ponto significa uma célula vazia
